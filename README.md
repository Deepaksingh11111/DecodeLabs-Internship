📁 Repository Structure 

project1.py: Deterministic Logic Engine — A highly optimized, continuous control loop rule-based chatbot.project2.py: Supervised Learning Pipeline — A modular, production-ready $K$-Nearest Neighbors ($KNN$) classification pipeline utilizing the Iris benchmark dataset.🤖 Project 1: Rule-Based Chatbot (Deterministic Logic Engine)🔬 OverviewProject 1 establishes a firm grasp on the core Input ➔ Process ➔ Output control sequence using clean, type-hinted Python syntax. By eliminating hardcoded "magic strings" and utilizing a hash map (dict) configuration, it guarantees direct intent matching with an efficient time complexity of $O(1)$.🛠️ Technical ArchitectureContinuous Control Loop: Operates an active runtime state that processes strings continuously until a dedicated KILL_COMMAND (exit) sequence is caught.Sanitization Pipeline: Normalizes text feeds by purging structural padding (whitespace) and modifying character cases to prevent mismatch errors.Atomic Fallbacks: Employs explicit mapping lookups via .get() to handle unexpected inputs gracefully without breaking runtime stability.🚀 How to RunBashpython project1.py
📊 Project 2: Iris Classification Pipeline (Supervised Learning)🔬 OverviewProject 2 transitions from strict rules into statistical pattern recognition. Built using object-oriented principles, this framework encapsulates data splitting, standardization geometry, distance-based training, and diagnostic metric evaluation within a unified pipeline.🛠️ Pipeline Stages[Stage 1: Input & Scale] ➔ [Stage 2: Proximity Fit] ➔ [Stage 3: Diagnostic Output]
Stage 1: Input & PreprocessingLoads the 150-sample balanced Iris dataset.Partitions data using a strict 80/20 train-test split, stratified to eliminate class representation bias.Applies a StandardScaler calculation matrix to bring all feature dimensions to a mean ($\mu$) of $0$ and a variance ($\sigma^2$) of $1$, optimizing distance calculations.Stage 2: Process (The Proximity Principle)Instantiates a $K$-Nearest Neighbors ($KNN$) model ($K=5$).Maps structural spatial clusters inside the training matrix.Stage 3: Output ValidationExposes pipeline accuracy through a structural Confusion Matrix (Pandas DataFrame).Generates precision, sensitivity (recall), and harmonic $F_1$-score classification reports.🚀 How to RunBashpython project2.py
